## 一、世界观

### 1.1 魔法岛（Island）

一座被大海环绕的神秘小岛，外面的人并不知道它的存在。岛上有一层**防护罩**保护着，平时看不见，只有在特殊时刻（台风、星光之夜）才会显现。

岛上的居民过着普通的生活，他们不知道魔法学院的城堡其实是一座真正的魔法学校——在他们眼里，那只是一栋漂亮的老房子。

**岛的地理分区（手绘地图锚点）**

| 分区 | 位置 | 代表角色 |
|---|---|---|
| **魔法城堡** | 中央山坡 | 晶晶、亮亮、老师们、同学 |
| **花园** | 城堡西侧 | 小昆虫、艾丽斯、小玫瑰 |
| **操场** | 城堡东侧 | 全员集合点；防光罩练习场 |
| **森林小路** | 城堡北侧 | 通向海边 |
| **海滩** | 岛的最北 | 塔利亚守护；小海洋的家 |
| **居民小镇** | 城堡外围 | 不知情的普通岛民（不知道城堡是魔法学院，以为只是一栋漂亮的老房子） |
| **海面** | 全岛环抱 | 塔利亚的领域 |

---

### 1.2 魔法学院（Academy）

建在魔法岛中央山坡上的古老城堡，**法式新哥特风格**：白墙蓝顶、尖塔、玫瑰花窗、彩色玻璃、金色雕花栏杆、白色大理石旋转楼梯。城堡里有水晶吊灯、飘浮的抱枕、说话的水晶球。

> **视觉锚点**：`assets/banners/城堡横版.jpg` / `城堡玩耍.jpg` / `城堡近景横版.jpg`

**班级划分**

- **初级班**：4–6 岁，学最基础的感知魔法
- **中级班**：5–12 岁（亮亮在这里，比晶晶早入学一点点）
- **高级班**：13–18 岁，学保护小岛的魔法

学院有魔法课、画画课、音乐课、甜点课……每一节课都很有趣。

---

### 1.3 国家守护者（Guardians）

四位守护者守护着魔法岛和防护罩的安全。他们平时很神秘，只有在小岛遇到危险时才会出现。

**视觉共性**：站姿是"单手叉腰 / 单手摊开魔法阵光"；身后有**半透明蛋壳形魔法阵光晕**；身穿长礼服 + 高跟鞋 + 王冠 + 守护杖。

| 姓名 | 守护元素 | 视觉特征 | 服装主色 |
|---|---|---|---|
| **奥利夏** | 阳光 | 粉紫渐变长卷发，王冠 | 粉紫渐变礼服，紫水晶杖 |
| **奥利安** | 星空 | 金色长马尾，红宝石杖 | 橙金长礼服 |
| **艾丽斯** | 花园 | 粉发双马尾，**粉色翅膀** | 粉白花瓣礼服，心形杖 |
| **塔利亚** | 海洋 | 浅蓝短发，蓝宝石杖 | 蓝白渐变礼服 |

> 奥利夏与奥利安名字呼应"夏夜星空"——奥利夏是白天的太阳守护者，奥利安是夜晚的星星守护者。

## 三、魔法体系（基础规则，不复杂）

### 3.1 入门魔法 · 感知

初级生最先学的是**感知魔法**——闭上眼睛，感受身边的魔法元素。

故事里晶晶第一次就感受到了一颗小光球；亮亮感受到了一朵棉花糖一样的小云。

### 3.2 魔法方向（每个小朋友各有所长）

| 方向 | 代表 |
|---|---|
| **光魔法** | 小光明、小太阳 |
| **月魔法** | 小月亮、小月光 |
| **自然魔法** | 小昆虫、艾丽斯 |
| **食物魔法** | 小甜点、小海绵阿姨 |
| **海洋魔法** | 小海洋、塔利亚 |
| **黑暗/影子魔法** | 小暗暗（其实是善意的） |
| **动物魔法** | 小喵喵、小汪汪 |

### 3.3 魔法道具

- **魔杖**：每人有专属形状（搅拌器、画笔、星星、月牙、心形……）
- **魔法铅笔**：普通铅笔进入学院后会自动获得名字（晶晶的叫"小橘子"、亮亮的叫"小星星"，还有"小太阳""小月亮"等）
- **防护罩**：保护小岛；**防光罩**是户外练习时的临时遮阳罩（像大帐篷）
- **敲墙**：宿舍隔墙信号系统（见 4.2）

### 3.4 魔法三规

1. 魔法只能用来帮助别人，不能用来欺负人
2. 不可以在学院外面随便使用魔法
3. 每天要按时睡觉，睡好了魔法才会更强

## 四、学院生活

### 4.1 一天的安排

| 时间 | 活动 |
|---|---|
| 早晨 | 起床、洗漱、去食堂吃早餐（**麦片粥**是 C 位） |
| 上午 | 魔法课（在教室或操场，晴天户外要撑防光罩） |
| 中午 | 食堂午餐 + 午睡 |
| 下午 | 兴趣课（画画、唱歌、做甜点……） |
| 傍晚 | 自由活动 / 散步 |
| 晚上 | 回宿舍、洗漱、敲墙、睡觉 |

### 4.2 宿舍与敲墙暗号（晶晶+亮亮专属）

姐姐和妹妹的房间**只隔一面薄薄的墙**（见 `卧室.jpg`：两床完全对称，中间隔墙伸手可触）。

晚上睡前，她们隔着墙传暗号：

| 敲几下 | 暗号 |
|---|---|
| 1 下 | 平安 |
| 2 下 | 晚安 |
| 3 下 | 我已经开始锻炼身体啦 |
| 4 下 | 我的能量已经到百分之一百啦！ |
| 5 下 | 姐姐撞墙了（太困了，头碰到墙） |
| 6 下 | 妹妹也撞墙了 |

> 📌 这是爸爸整理时女儿口述的"原味细节"，第一集末尾要用上。

### 4.3 食堂魔法

- **影子**会自动飘起来帮忙拿东西
- 麦片粥会自己跑到碗里
- 餐桌会根据每个人的口味变出想吃的菜
- 厨房里的彩色魔法锅（`厨房.jpg`）会自己煮饭、盘子飘在空中
