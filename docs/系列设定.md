## 一、世界观

### 1.1 魔法岛（Island）

一座被大海环绕的神秘小岛，外面的人并不知道它的存在。岛上有一层**防护罩**保护着，平时看不见，只有在特殊时刻（台风、星光之夜）才会显现。

岛上的居民过着普通的生活，他们不知道魔法学院的城堡其实是一座真正的魔法学校——在他们眼里，那只是一栋漂亮的老房子。

**岛的地理分区（手绘地图锚点）**

| 分区 | 位置 | 代表角色 |
|---|---|---|
| **魔法城堡** | 中央山坡 | 晶晶、亮亮、老师们、同学 |
| **花园** | 城堡西侧 | 小昆虫、爱丽丝、小玫瑰 |
| **操场** | 城堡东侧 | 全员集合点；防光罩练习场 |
| **森林小路** | 城堡北侧 | 通向海边 |
| **海滩** | 岛的最北 | 塔利亚守护；小海洋的家 |
| **居民小镇** | 城堡外围 | 不知情的普通岛民（不知道城堡是魔法学院，以为只是一栋漂亮的老房子） |
| **海面** | 全岛环抱 | 塔利亚的领域 |

---

### 1.2 魔法学院（Academy）

建在魔法岛中央山坡上的古老城堡，**法式新哥特风格**：白墙蓝顶、尖塔、玫瑰花窗、彩色玻璃、金色雕花栏杆、白色大理石旋转楼梯。城堡里有水晶吊灯、飘浮的抱枕、说话的水晶球。

> **视觉锚点**：`assets/banners/城堡横版.jpg` / `城堡玩耍.jpg` / `城堡近景横版.jpg`

**班级划分**

- **初级班**：4–5 岁，学最基础的感知魔法（晶晶、亮亮在这里，亮亮比晶晶早入学一点点）
- **中级班**：6–7 岁，学更复杂的元素魔法
- **高级班**：8–9 岁，学保护小岛的魔法

> 📌 学院规模小，实行**混龄同班**：初级/中级/高级是魔法课程的进度分级，不是分班依据——同班、同桌可以跨级（v4 设定）。

学院有魔法课、画画课、音乐课、甜点课……每一节课都很有趣。

### 1.3 国家守护者（Guardians）

四位守护者守护着魔法岛和防护罩的安全。他们平时很神秘，只有在小岛遇到危险时才会出现。

**视觉共性**：站姿是"单手叉腰 / 单手摊开魔法阵光"；身后有**半透明蛋壳形魔法阵光晕**；身穿长礼服 + 高跟鞋 + 王冠 + 守护杖。

| 姓名 | 守护元素 | 视觉特征 | 服装主色 |
|---|---|---|---|
| **奥利夏** | 阳光 | 粉紫渐变长卷发，王冠 | 粉紫渐变礼服，紫水晶杖 |
| **奥利安** | 星空 | 金色长马尾，红宝石杖 | 橙金长礼服 |
| **爱丽丝** | 花园 | 粉发双马尾，**粉色翅膀** | 粉白花瓣礼服，心形杖 |

## 三、魔法体系（基础规则，不复杂）

### 3.1 入门魔法 · 感知

初级生最先学的是**感知魔法**——闭上眼睛，感受身边的魔法元素。

故事里晶晶第一次就感受到了一颗小光球；亮亮感受到了一朵棉花糖一样的小云。

### 3.2 魔法方向（每个小朋友各有所长）

| 方向 | 代表 |
|---|---|
| **光魔法** | 小光明、小太阳 |
| **月魔法** | 小月亮、小月光 |
| **自然魔法** | 小昆虫、爱丽丝 |
| **食物魔法** | 小甜点、小海绵阿姨 |
| **海洋魔法** | 小海洋、塔利亚 |
| **天气魔法** | 小天天 |
| **黑暗/影子魔法** | 小暗暗（其实是善意的） |
| **动物魔法** | 小喵喵、小汪汪 |
| **石头魔法（待揭晓）** | 茉茉（第六集石头魔杖已醒、亮成茉莉色，但"魔法是什么"仍未揭晓——茉茉和安雨约定"自己想一想"，留待第七集） |
| **龙族魔法** | 安雨（**龙族展翅**：红金色半透明光之龙翼，平时背上没有翅膀，展翅时光从肩膀长成翅膀） |

### 3.3 魔法道具

- **魔杖**：每人有专属形状（搅拌器、画笔、星星、月牙、心形……）
- **晴天娃娃**（小天天专属）：白圆脸、粉腮红、淡紫头发的小魔法娃娃，是小天天施展天气魔法的媒介。它白天乖巧地趴在主人肩上，夜晚会自己抬头收集月光。一旦找不到它——或者小天天找不到它——小天天就会因为"借不出天气"而情绪失控，**她的心情直接变成天气：开心出太阳、难过下雨、害羞出彩虹**（v2 设定，由小橘子口述"小天天找不到晴天娃娃后就一直哭，然后就一直下雨"确立）。
- **甜心娃娃**（小甜心专属）：白布小头、淡粉色短发、黄色眼睛、穿紫色裙子（胸口有爱心图案）的布娃娃，是小甜心最喜欢的娃娃，睡觉/吃饭/上课都要抱着。小甜心的情绪通过它"漏"到外界——**她一难过，外面的世界就会褪色：春天变回秋天（树叶黄、草地黄、花垂头）；她一开心，世界会开成比春天更春天（花更粉更亮）**。只影响"颜色和植物"，不真的变冷（v3 设定，由小橘子口述"小甜心流了一滴眼泪，春天变回了秋天"确立；娃娃形象以 `assets/characters/小甜心抱甜心娃娃.png` 为准）。
- **玩偶伙伴**（晶晶+亮亮专属）：晶晶的**黑黑**（全黑色玩偶猫，金眼睛）和亮亮的**奶 ki**（三花色玩偶猫：橘/白/灰，原型是小橘子现实中的两个玩偶，"奶 ki"的 ki 按正常拼音规则发音）。依据"在魔法学院，所有东西都有自己的名字"的规则，玩偶在学院里会活过来。黑黑的魔法是"躲猫猫"——往阴影里一趴，连它抱着的东西一起隐身；奶 ki 的魔法是"变软变漂亮"——身上的三种颜色能变成丝带、包装纸、蝴蝶结（v3 设定，由小橘子口述"晶晶和亮亮用黑黑猫和奶 ki 猫的魔法给小甜心惊喜"确立）。
- **石头魔杖 + 小彩虹**（茉茉专属）：石头魔杖是小包子老师特地为茉茉准备的入学礼物，杖顶是一颗圆圆的灰石头——魔杖"在睡觉"，等认出小主人才会亮（v5 伏笔：第四集结尾，茉茉房间的门缝透出一丝**茉莉色的光**，是魔杖醒了还是小彩虹醒了，留待后续揭晓；第五集进展：茉茉与安雨约定两根睡觉的魔杖"一起睡，醒了以后一起亮"；v7 进展：第六集清晨在花园练习时，石头魔杖醒了——灰石头亮成淡淡的茉莉色，石头正中间像开了一朵小小的茉莉花（兑现"一起睡，一起亮"；但"魔法是什么"两人决定"自己想一想"，待第七集）。
- **龙珠魔杖 + 小红豆**（安雨专属）：龙珠魔杖是小包子老师特地为安雨准备的入学礼物——红色杖身，杖顶金色龙首，龙嘴含一颗圆珠。魔杖跟着安雨离开很远很远的家，"飞累了"，一直在睡觉（呼噜呼噜像打哈欠），**等它认出这里是新家就会醒**（v6 伏笔：第五集结尾，安雨房间门缝透出一丝**红色的微光**，像珠子发出来的光；v7 进展：第六集与石头魔杖同时醒来——龙嘴里的珠子亮成红色，暖暖的像一颗小太阳，不再打呼噜，认出这里是新家了）。**小红豆**是安雨的龙女娃娃（龙鳞红裙、黑发双丸子头、额头金色龙纹，形象以 `assets/characters/安雨抱小红豆.jpg` 为准），安雨让她陪着魔杖睡觉——"有人陪着，就不会做噩梦了"（v6 新增，名字由爸爸+小橘子共创，取"红豆相思"想家之意）。**小彩虹**是茉茉的独角兽玩偶（白白身子、粉粉卷毛、金色小角），茉茉想家时寸步不离地抱着；按"学院里所有东西都会活过来"的规则，晶晶好像看到它眨过眼睛（待证实）。
- **魔法铅笔**：普通铅笔进入学院后会自动获得名字（晶晶的叫"小橘子"、亮亮的叫"小星星"，还有"小太阳""小月亮"等）
- **防护罩**：保护小岛；**防光罩**是户外练习时的临时遮阳罩（像大帐篷）
- **敲墙**：宿舍隔墙信号系统（见 4.2）

### 3.4 魔法三规

1. 魔法只能用来帮助别人，不能用来欺负人
2. 不可以在学院外面随便使用魔法
3. 每天要按时睡觉，睡好了魔法才会更强

### 4.5 小天天的天气魔法（v2 新增 · 由小橘子口述）

天气魔法是**情绪型魔法**——小天天的心情直接对应头顶的天气，全学院都能"看到"她的情绪：

| 小天天的状态 | 天气 |
|---|---|
| 开心、找到晴天娃娃 | ☀️ 阳光 |
| 难过、找不到晴天娃娃 | 🌧️ 一直下雨 |
| 害羞、被夸奖 | 🌈 彩虹 |
| （待补充）生气 | ⚡ 打雷？ |

**晴天娃娃丢失协议**（学院默认反应）：
- 一旦发现下雨不止 + 小天天不在 → 大家先集合，再分组去城堡各角落找娃娃
- 找的过程中可以用自己的魔法帮忙"撑住"（如晶晶用水晶保护罩挡雨）
- 但**不要用强魔法直接覆盖天气**——那只是治标，不解决小天天的心情

> 📌 这个设定 v2 确立，第二集《晴天娃娃不见了》是它的首秀。后续可以发展：小天天学会更精细地控制天气（不下雨只阴天）、晴天娃娃自己会"离家出走"找乐子、其他人也想有能映射心情的魔法道具……

### 4.6 小甜心的季节调色魔法（v3 新增 · 由小橘子口述）

季节调色魔法也是**情绪型魔法**——小甜心的心情直接对应外界的"季节色调"，比天气魔法更柔和（只影响颜色和植物，不真的变冷）：

| 小甜心的状态 | 外界 |
|---|---|
| 开心、抱着甜心娃娃 | 🌸 春天，花更粉更亮 |
| 难过、甜心娃娃不见 | 🍂 一滴眼泪→春天褪色成秋天（叶黄草黄花垂头） |
| 开心到又哭又笑 | 🌸✨ 比春天更春天 |

**甜心娃娃丢失协议**（学院默认反应）：
- 小甜心不会嚎啕大哭，她会"忍住，坚持找"——但她越忍，褪色越厉害
- 不要用强魔法直接覆盖季节——重点是让小甜心真正笑出来
- 如果褪色发生在夜里，优先确认是不是有人借走了甜心娃娃（惊喜待定）

> 📌 v3 确立，第三集《甜心娃娃不见了》是它的首秀。和天气魔法并列，构成"情绪外溢型魔法"的第二种形态。后续可以发展：小甜心学会只褪色一小块地方（比如只让花园变秋天）、甜心娃娃自己会"脸红"（爱心图案变亮）、小玫瑰和小甜心的"谁更粉"比赛升级……
